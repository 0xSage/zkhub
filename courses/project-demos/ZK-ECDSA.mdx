# ZK-ECDSA + Groebner Basis for circuit testing

Efficient implementations of secp256k1 cryptographic functions in circom, enabling SNARKs for Ethereum identity primitives.

Also, a proof of concept demonstration for how to use multivariate polynomial solvers to test ZK circuit constraint soundness.
